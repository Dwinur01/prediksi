# Template Import Pasien

File template ini dapat digunakan untuk mengimpor data pasien secara massal ke dalam sistem.

## Daftar Kolom
Berikut adalah daftar kolom beserta format yang diharapkan dalam file Excel:

| Nama Kolom | Keterangan | Contoh Nilai |
| :--- | :--- | :--- |
| **NIK** | Nomor Induk Kependudukan (Wajib) | 35251100001 |
| **Nama** | Nama Lengkap Pasien | Budi Santoso |
| **Jenis Kelamin** | Laki-laki / Perempuan | Laki-laki |
| **Tanggal Lahir** | Format YYYY-MM-DD | 1978-05-12 |
| **Alamat** | Nama Desa atau Alamat Lengkap | Cerme Lor |
| **No HP** | Nomor Telepon Aktif | 081234567890 |
| **Sistolik** | Tekanan Darah Sistolik (mmHg) | 145 |
| **Diastolik** | Tekanan Darah Diastolik (mmHg) | 95 |
| **IMT** | Indeks Massa Tubuh | 26.5 |
| **Merokok** | Ya / Tidak | Ya |
| **Alkohol** | Ya / Tidak | Tidak |
| **Sayur/Buah** | Konsumsi Sayur & Buah (Ya/Tidak) | Ya |
| **Diabetes** | Memiliki Diabetes (Ya/Tidak) | Tidak |
| **Keluarga Hipertensi** | Riwayat Hipertensi Keluarga (Ya/Tidak) | Ya |

### Petunjuk Penggunaan
1. Unduh template Excel yang telah disediakan.
2. Isi data pasien sesuai dengan format pada tabel di atas.
3. Simpan file sebagai `.xlsx`.
4. Unggah file melalui menu **Import Excel** di halaman Data Pasien.
